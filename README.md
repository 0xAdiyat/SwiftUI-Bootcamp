#### Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/0xAdiyat/SwiftUI-Playground.git
   ```
2. Open the Xcode project:
   ```bash
   cd SwiftUI-Bootcamp
   open SwiftUI-Bootcamp.xcodeproj
   ```
4. Run the app in Xcode.
